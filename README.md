## Download & install

1. Grab the latest **`Bezel-0.1.0-Setup.exe`** from [**Releases**](https://github.com/BezelMedia/Bezel/releases/latest).
2. Run it. Windows SmartScreen may warn that the app is unrecognized (it isn't code-signed yet) — click **More info → Run anyway**.
3. On first launch Bezel sets up its folders under `Documents\Bezel` and downloads RetroArch from libretro in the background. Drop your games into the per-system folders under `Documents\Bezel\roms` and hit **Scan**.

> Prefer not to install? The **Portable** build runs without installing.

## Bring your own games

Bezel ships **no ROMs and no BIOS** — you supply your own. It identifies your files from a built-in offline database (No-Intro / Redump), pulls box art automatically, and never phones home for your library.

## Features

- 🖼️ **Visuals-first shelf** — box art, hero banners, descriptions, facts; browse every title even ones you don't own.
- 🎮 **In-app play** (built-in cores) or **RetroArch** (downloaded on first run) for disc and heavy-3D systems.
- 💿 **Discs** — identify, rip, or play straight off the drive.
- ☁️ **Cloud-synced saves** via a folder your own Drive/Dropbox/OneDrive keeps synced.
- 🌐 **Live on-screen translation** for import games.
- 🕹️ **Controller-first** navigation throughout.
- 📴 **Offline & private** — the game database is built in; no account, no tracking.
